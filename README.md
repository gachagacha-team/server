# 👾 가챠가챠 👾

## 프로젝트 소개

- 가챠가챠(GachaGacha)는 아이템 뽑기 및 거래, 미니홈 꾸미기 등의 기능을 제공하는 커뮤니티 플랫폼입니다.

## 사용 기술

### Backend

- Java
- Spring Boot
- JPA/Hibernate
- Docker

### Database

- MySQL
- Redis

### Infra

- AWS ec2, rds
- Docker
- Github Actions
